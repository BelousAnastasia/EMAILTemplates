USB 3.0 Drives - Enhanced Data Transfer Rate
Boosted Data Transfer Speed for USB 3.0 Drives in USB Network Gate 9.1

If you regularly share USB drives over the network, then you definitely should install the latest update! USB Network Gate 9.1 significantly improves the data transfer rate for USB 3.0 hard drives: a whooping x4 increase in speed on the same network when copying files from USB 3.0 drives.
Update Now

Some more good news: it is a “minor” update, so all current users of USB Network Gate 9.0 can enjoy it right away and for free.

To enjoy the benefits of this update,
please make sure:
  • A USB drive is connected to a computer’s USB 3.0 port.
  • Server OS: Windows 7 and above, Windows Server 2008R2 and above*. 
  • Client OS: Windows 10, Windows Server 2016/2019.

* Windows 7 and Windows Server 2008 R2 servers might not display improved performance with certain USB controllers.
